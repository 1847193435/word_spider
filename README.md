# word_spider
一个使用webmagic框架的爬取单词入门案例

# 技术栈
- spring
- mybatis
- mybatis 逆向工程
- webmagic
- spring整合mybatis

# 思路
使用webmagic爬取 **http://word.iciba.com/** 上的四级单词（这里只爬取了四级单词，其他单词参照示例），爬取到的数据通过mybatis加载到数据库中
